##### このリポジトリについて  

###### このリポジトリの概要と目的
デザインパターンの問題&解答集。  
個人的に重要・よく使うと判断したパターンを抜粋して掲載している。
この問題集を繰り返し解いて、デザインパターンを手で覚えることが目的。

###### フォルダ構成  
* Questions  
  デザインパターンの問題集。    
  個人的に重要なパターンに絞って掲載。  
* AnswerPhp  
  PHPによる解答集。
